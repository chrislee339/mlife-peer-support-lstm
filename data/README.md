# Data

The mLIFE trial data is protected health information and is not included in this repository.

The cleaned panel used by the analysis pipeline is `data/cleaned/weight_windows_future_targets_all_weeks.csv` (and the per-window/stride variants under `data/cleaned/sweeps/`). These are reconstructed from the raw mLIFE trial extracts using `code/sweep_rebuild_dataset.py`.

Access to the underlying trial data requires data-use agreement approval from the mLIFE trial coordinating site. Contact the corresponding author or the mLIFE primary trial team (Turner-McGrievy et al., 2025) for access requests.
